# Real-Estate Talking-Head Thumbnail Concept

![Buy Now? Rates Are Up thumbnail](buy-now-rates-are-up.png)

This is a **self-initiated fictional concept**, not commissioned client work. It demonstrates an urgent, approachable YouTube thumbnail direction for first-time buyer, mortgage-rate, or housing-policy videos aimed at adults roughly 22-38 years old.

The fictional talking-head presenter, house, and rate-arrow graphic support a short, mobile-readable question: `BUY NOW?` The asset deliberately includes no client name, real-estate agency, real listing, celebrity likeness, analytics, CTR, conversion, or performance claim.

## Delivery specification

- 1280 x 720 pixels (16:9)
- PNG, RGB
- Large contrast-led headline with safe margins
- Fictional presenter and property; no logos or brand identifiers

## Production note

The core visual was generated with OpenAI's built-in image generation tool and then resized with Lanczos resampling to the final 1280 x 720 delivery. See [SOURCES.md](SOURCES.md) for asset provenance and commercial-use notes.

For an actual client project, the video premise, supplied or licensed presenter photo, geographic/policy context, brand styling, factual claims, revision count, final dimensions, and delivery usage would be agreed in writing before production.
