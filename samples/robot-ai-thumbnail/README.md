# Robot AI YouTube Thumbnail

A 1280 x 720 YouTube thumbnail concept for the title **"I gave ChatGPT a Body... & Here's What Happened"**.

The design turns the supplied rover camera into the focal point and compresses the story into a fast mobile-size hook: **IT TOOK CONTROL**. The supplied rover photography remains the dominant subject; all typography, layout, grid, reticle, and callout graphics were composed manually in Pillow.

This is a spec test made in response to a public hiring form. It is not client work and makes no CTR, revenue, or performance claim.

## Files

- `robot-ai-thumbnail.png` - final 1280 x 720 RGB export
- `build.py` - deterministic composition script
- `source/` - two rover images supplied by the form owner
- `SOURCES.md` - source and usage notes

## Rebuild

```powershell
python build.py
```
